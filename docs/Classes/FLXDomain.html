FLXDomain

@interface FLXDomain : NSObject <FLXDomainProtocol>

Main FLUX class that is designed to be subclassed by your application domain.

Warning

This class is designed for subclassing by your application domain, but be careful with overriding base implementation of FLXDomainProtocol methods. Recommended way to subclass a domain is to extend it’s implementation with more convinient methods e.g. specific store accessors, constructors etc.
